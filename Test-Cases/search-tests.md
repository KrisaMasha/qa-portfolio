# Тест-кейсы: Проверка поисковой системы centerfish.ru

## TC-001: Поиск по точному названию "катушка"
**Priority:** High  
**Preconditions:** Открыта страница поиска

### Steps:
1. Ввести в поисковую строку: "катушка"
2. Нажать Enter/кнопку поиска

### Expected Result:
Система находит товары, содержащие точное совпадение слова "катушка"
