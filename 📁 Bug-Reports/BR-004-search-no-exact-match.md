# BR-004: Поиск не находит товары по точному названию "чехлы"

**Project:** CenterFish.ru  
**Related Test Case:** [TC-002](../Test-Cases/search-tests.md#tc-002)  
**Session ID:** [RUN-003](../Test-Runs/RUN-003-search-functional.md)  
**Severity:** Medium  
**Priority:** High  
**Environment:** Chrome 120.0, Windows 10  
**Date Reported:** 11.03.2026  
**Status:** Open

---

## 📝 Описание
Система отображает "Не найдено ни одного товара" при поиске по слову "чехлы", хотя товары с таким названием существуют в каталоге.

## Preconditions
- Открыта страница поиска

## 🔄 Steps to Reproduce
1. Ввести в поисковую строку: "чехлы"
2. Нажать поиск

## ✅ Expected Result
Система находит товары, содержащие слово "чехлы"

## ❌ Actual Result
Система отображает "Не найдено ни одного товара", хотя товары с таким названием существуют в каталоге

## 📎 Attachments
- Скриншот: https://skr.sh/saTlgg9VnH1

## 💡 Impact
Пользователи не могут найти товары категории "чехлы", что приводит к потере продаж.

## 🔧 Suggested Fix
Проверить индексацию товаров в поисковой системе и корректность работы алгоритма поиска по точному совпадению.
